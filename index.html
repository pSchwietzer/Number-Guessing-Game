Guess a Number

In this game you have to guess a number between a set range. You can guess 3 times.

Controls

Info screen

Arrow Keys

Switch between difficulties

Enter

Start the game

Game

0-9

Enter numbers to input

Enter

Submit guess / Restart game

Backspace

Remove last character

Delete

Delete all characters

Escape

Close game and return to info

Select Difficulty

Guess a random number between 0 and 10

You can guess 3 more times!
